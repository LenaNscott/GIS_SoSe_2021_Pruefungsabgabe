Das digitale Memory

Wie funktioniert das digitale Memory?

Im Menü oben rechts kommt man zu den jeweiligen Seiten.

Über Score kommst du auf die High-Score Tabelle, wo die besten 10 aufgelistet sind.

Von dort aus kann man ein neues Spiel starten.

Über Admin kommst du zu den Bildern welches das Memory benutzt.

Wenn du möchtest kannst du dort Bilder löschen oder du kannst neue Bilder mit einer URL hinzufügen.

Durch das Klicken des "Spiel starten" Buttons kommst du auf die Spielseite.

Dort liegen 4x4 umgedrehte Karten, sobald du auf start klicks startet die Uhr.

Sind zwei angeklickte Karten gleich, verschwinden sie, ansonsten werden sie wieder umgedreht.

Danach klickst du wieder zwei Karten an bis alle Karten von dem Spielfeld verschwunden sind.

Achtung die Zeit und die Anzahl der klicks sind wichtig.

Ist das Spiel zu Ende wirst du auf eine Seite weitergeleitet welche deine Zeit anzeigt und du deinen Namen eingeben kannst.
